1. Basic notes

1.1 Values and Types

BoaScript is a dynamically typed language. There are no type definitions. There are two types in BoaScript: numerical (num) and string.

1.2 Expressions

The basic expressions in BoaScript are the following: 


        expr ::= num
        expr ::= string

1.3 Output

BoaScript has two output operators: 


        print   expr; - prints the expr
        println expr; - same as previous but adds CR/LF at the end

Example: 


        println 2;
        print sin(a) + pow(2, 4);
        println "Hello sweet victory";

1.4 Comments

Line or part of line will be ignored by the interpreter from double slash to the end of line.

Example: 


        // This is comment.
        a = 2; // Assignment.
        print a;
        // End of script.

2. Variables.

Variables are places that store values. BoaScript has 26 one-character names reserved for variables - a~z. 


       var ::= expr

Example: 


        a = 2;
        b = sin(a) + pow(2, 4);
        s = "Hello sweet victory";

3. Statements

BoaScript supports a set of statements, similar to those in C. This set includes assignments, control structures, and variable declarations.

3.1 Chunks

Chunk is a unit of BoaScript execution. A chunk is a sequence of statements, which are executed sequentially. Each statement is followed by a semicolon: 


        chunk ::= { stat; }

3.2 Blocks

A block is a list of statements; syntactically, a block is the same as a chunk: 


        chunk ::= chunk

3.3 Assignment

Assignments in BoaScript are same as in C. 


        var ::= num;
        var ::= string; 
        var ::= var;
        var ::= expr;

3.4 Control Structures

The control structures if, while, and ? have the usual meaning and familiar in C syntax: 


        val  ::= expr ? expr : expr
        stat ::= while (expr) block
        stat ::= if (expr) block [else block]

Example: 


        s = "A" > "B" ? "Yes" : "No"; // Operator ?
        b = 1 < 2;
        a = b ? 5 : 10;
         
        // Operator "if"
        if (a < 2) 
        {
            b = sin(c) + cos(d); 
        }
        else
        {
            b = 12345.34e-1;
        }
        
        // or
        if (a == 2) { b = 3; }
        
        // Operator "while"
        while (x < 10)
        {
            y = x ^ 2;
            ++x;
        }

3.5 Arithmetic operations

BoaScript supports the usual arithmetic operators: the binary + (addition), - (subtraction), * (multiplication), / (division), % (modulo), and ^ (exponentiation); and unary - (negation).

Example: 


        b = 1 + 2 * (4.5 - 8.9) / 3.14 + 5 % 2;
        a = 2 ^ (1 /6);

3.6 Relational operators

The relational operators in BoaScript are: 


        ==    !=    <     >     <=    >=

The result of this operations is "1" if true or "0", if false. Example:

Example: 


        a = 2;
        b = 3;
        println a > b;
        println a < b;
        c = (a == b) + 1;
        
        x = "Hello";
        y = "Hella";
        z = (x < y) ? 5 : 10;
        
        print "A" == "A";
        a = "A" > "B";

3.7 Logical operators

The logical operators in BoaScript are: 


        ||    &&

Example: 


        a = 2;
        b = 3;
        c = 4;
        println a || b && c;

3.8 Bit operators

The operators in BoaScript are having the same meaning as in C: 


        |    &    !     <<     >>

Example: 


        print 1 << 2;
        print !1;

3.9 Concatenation

The string concatenation operator in BoaScript is easy:

Example: 


        s = "Hello";
        t = " world";
        print s + t;

4. Functions

BoaScript has a number of embedded functions.

4.1 Functions returning numerical value

abs(num) Absolute value of number
acos(num) Arc cosine function
asin(num) Arc sine function
atan(num) Arc tangent function
atan2(num, num) Arc tangent function of two variables
ceil(num) Smallest integral value not less than argument
cos(num) Cosine function
cosh(num) Cosine hyperbolic function
exp(num) Exponent function
fabs(num) Absolute value of number
floor(num) Largest integral value not greater than argument
fmod(num, num) Floating-point remainder function
frexp(num, num) Convert number to fractional and integral components
log(num) Natural logarithm function
log10(num) Logarithm of base 10 function
max(num, num) Maximal value among two expressions
min(num, num) Minimal value among two expressions
modf(num, num) Extract signed integral and fractional values
pi() Returns pi-number
pow(num, num) Power function
rand() Random number generator in the range [0,1]
sin(num) Sine function
sinh(num) Sine hyperbolic function
sqrt(num) Square root function
strlen(string) Returns length of string
tan(num) Tangent function
tanh(num) Tangent hyperbolic function
tonum(string) converts string to number

Example: 


        print sqrt(2);
        a = sin(sqrt(cos(0) + 1)) + 
            atan(sin(3.1415926 / 2));
        b = rand();
            
        a = strlen("Hello world");
        print strlen("AAAAAAAA");
            
        a = tonum("1234567");
        print a;

4.2 Functions returning string

substr(string, num, num) Substring defined by indexes
replace(string, string, num, num) Replase part of string by another
tostring(num) Convert number to string
date(string) Return date or/and current time
substitute(string, string, string) Substitute target string instead the pattern

Example: 


        s = "Hello";
        print substr(s, 0, strlen(s) - 2);
       
        s = "Hello";
        t = replace(s, "pp", 2, 2);
           
        s = tostring(1234567);
        print s;
            
        s = date("dd-mm(mmm)-yy(yyyy) hh:MM:SS");
        print s;
        s = date("mm/dd/yy hh:MM");
        print s;
        //where
        //   d - day of month [0...31];
        //   m - month [1...12] (mm format) or abbrev. of month (mmm format);
        //   y - year (two digits or four digits);
        //   h - hour of day [1...12];
        //   H - hour of day [0...23];
        //   M - minute [0...59];
        //   S - second [0...59];
           
        a = "Hello"; 
        s = substr(a, 0, 3);
        t = "BoaScript";
        u = s + " " + t;

4.3 Special Functions

Example: 


        val = ifn(expr1, expr2, expr3);
        val = ifs(expr1, expr2, expr3);
        
        Returns expr2 if expr1 is true or expr3 otherwise.

Example: 


        s = ifs(1 < 2, "Yes", "No"); // ifs(num, string, string)
        t = ifs("A" >= "B", "True", "False");
